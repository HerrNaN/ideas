Rewriting Rules

Implication elimination

   f1 → f2   ⇔ (¬ f1) ∨ f2

Equivalence elimination

   f1 ↔ f2   ⇔ (f1 ∧ f2) ∨ ((¬ f1) ∧ (¬ f2))

The Morgan

   ¬ (f1 ∧ f2)   ⇔ ¬f1 ∨ ¬f2

The Morgan

   ¬ (f1 ∨ f2)   ⇔ ¬ f1 ∧ ¬ f2

Double negation

    ¬ ¬f   ⇔ f

Distributivity

   f1 ∧ (f2 ∨ f3)   ⇔ (f1 ∧ f2) ∨ (f1 ∧ f3)

Distributivity

   f1 ∨ (f2 ∧ f3)   ⇔ (f1 ∨ f2) ∧ (f1 ∨ f3)

Idempotence

   f1 ∧ f1   ⇔ f1

Idempotence

   f1 ∨ f1   ⇔ f1

Associativity

   (f1 ∧ f2) ∧ f3   ⇔ f1 ∧ (f2 ∧ f3)

Associativity

   (f1 ∨ f2) ∨ f3   ⇔ f1 ∨ (f2 ∨ f3)

Communativity

   f1 ∧ f2   ⇔ f2 ∧ f1

Communativity

   f1 ∨ f2   ⇔ f2 ∨ f1

False-True

   T ∨ f   ⇔ T

False-True

   T ∧ f   ⇔ f

False-True

   F ∧ f   ⇔ F

False-True

   f ∨ ¬f   ⇔ T

False-True
