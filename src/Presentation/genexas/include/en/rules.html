Herschrijfregels

Implicatie eliminatie

   f1 → f2   ⇔ (¬ f1) ∨ f2

Equivalentie eliminatie

   f1 ↔ f2   ⇔ (f1 ∧ f2) ∨ ((¬ f1) ∧ (¬ f2))

De Morgan

   ¬ (f1 ∧ f2)   ⇔ ¬f1 ∨ ¬f2

De Morgan

   ¬ (f1 ∨ f2)   ⇔ ¬ f1 ∧ ¬ f2

Dubbele negatie

   ¬ ¬f   ⇔ f

Distributiviteit

   f1 ∧ (f2 ∨ f3)   ⇔ (f1 ∧ f2) ∨ (f1 ∧ f3)

Distributiviteit

   f1 ∨ (f2 ∧ f3)   ⇔ (f1 ∨ f2) ∧ (f1 ∨ f3)

Idempotentie

   f1 ∧ f1   ⇔ f1

Idempotentie

   f1 ∨ f1   ⇔ f1

Associativiteit

   (f1 ∧ f2) ∧ f3   ⇔ f1 ∧ (f2 ∧ f3)

Associativiteit

   (f1 ∨ f2) ∨ f3   ⇔ f1 ∨ (f2 ∨ f3)

Communativiteit

   f1 ∧ f2   ⇔ f2 ∧ f1

Communativiteit

   f1 ∨ f2   ⇔ f2 ∨ f1
