Documentatie Exercise assistent online

1.- Installeer Apache met php

1.1

Op een windows machine gaat dat het gemakkelijkst met xampp: http://http://www.apachefriends.org/en/xampp.html.

1.2

In de php.ini in de apache/bin directory van xampp de bij de regel ;extension=php_curl.dll de puntkomma weghalen.

Als apache al is geïnstalleerd, met werkende php engine, kan dit worden overgeslagen.

2. - Webserver inrichten

2.1

xampp starten. Als je firewall vraagt of hij de poort moet openzetten: dat hoeft niet, omdat je alleen maar lokaal op je machine blijft.

mysql is niet nodig: alleen maar apache en php moet draaien.

2.2

Maak een directory feedback aan in de htdocs-directory van xampp (of in de directory die je webserver gebruikt).

Zet daarin: de hele inhoud van de directory feedback (dus inclusief alle directories).

3.- Happs draaien

3.1

Haal Happs 0.84 op, en configureer, build en installeer het met

runghc Setup.hs configure
runghc Setup.hs build
runghc Setup.hs install

Happs is te vinden op http://happs.org/HAppS/README.html#download.

Ik had ghc 6.6 nodig, en wanneer je runghc Setup.hs configure draait in de Happs-directory wordt er wel gemeld wat je allemaal nog meer nodig hebt.
Dat is ondermeer fps 0.8 dat alleen te krijgen is via darcs get --partial http://www.cse.unsw.edu.au/~dons/code/fps

3.2

Zet alles uit de hapss prog directory van de zip in een directory van waaruit je de happs server gaat draaien. Dat hoeft dus niet ergens binnen de directory van de apache webserver te zijn).

3.3

Compileer FeedbackEngine.hs.

3.4

Run de executable.

3.- De Exercise assistent on-line

3.5

Nu kun je in een browser http://localhost/feedback/ invoeren.

Als je een andere directory gebruikt dan de feedback directory, moet in javascript/clientfeedback.js worden veranderd:

http://localhost/feedback/math/genereer.php
en http://localhost/feedback/math/getfeedback.php
